# Calculator Application

A simple JavaScript calculator that performs basic arithmetic operations and stores a history of calculations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Stores calculation history
- Displays previous calculations
- Accounts for division by zero
- Accounts for non-numeric input values

## How to Run

1. Make sure Node.js is installed.
2. Open the project folder in your terminal.
3. Run:

```
node index.js
```

## Example Usage

Example function calls inside `index.js`:

```javascript
addNumbers(1, 2);
multiplyNumbers(2,5);
displayHistory();
```

Example Output:

```
Calculation History:
1. 5 + 3 = 8
2. 4 * 2 = 8
```
